SWITCH CASE

Estrutura de controle de fluxo
Avalia uma expressão
Procura a melhor cláusula(case/caso) que atende ao resultado

ANOTAÇÃO IMPORTANTE

Lembre-se o SWITCH CASE não para de executar o codigo até encontrar o BREAK.

Então quando ele identifica o case, se ele nao tiver um break e o case de baixo tiver, ele executará todo o codigo do case de baixo até encontrar o break.

Tome cuidado com isso, mas se for criativo vc tbm pode utilizar ao seu favor!!!

No caso/case DEFAULT não precisa de BREAK.
É como se ele tivesse um break por padrão dentro dele.
Ou seja se nenhum CASE tiver BREAK o codigo vai executar até o DEFAULT
