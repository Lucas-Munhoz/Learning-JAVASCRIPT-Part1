                    //declarando a função
                    function algumaTarefa() {

                    }

                    algumaTarefa()

PARÂMETRO DA FUNÇÃO

PARÂMETRO servem para passar valores para a função utilizar dentro do bloco de código dela. Da pra usar mais de um, só separar por virgula

                
                    //declarando a função
                    function algumaTarefa(PARÂMETRO) {
                        < bloco de código >
                    }

                    //chamando a função
                    algumaTarefa(PARÂMETRO)

PARÂMETRO PADRÃO

Caso a função não receba um parâmetro ela usará esse padrão que será passado na declaração

                
                    //declarando a função
                    function algumaTarefa(PARÂMETRO1, PARÂMETRO2 = "padrão") {
                        < bloco de código >
                    }

                    //chamando a função sem um dos parâmetros
                    algumaTarefa(PARÂMETRO1)

CUIDADOS COM OS PARÂMETROS

  1. Parâmetros não são necessários para executar uma função, mas se ela tiver algum que não tenha um "parametro padrão" ele ficará como "UNDEFINED"
  2. Cuidado com a ordem dos parâmetros, pois o "PARÂMETRO PADRÃO" não substitui o lugar na ordem dos parâmetros. EXEMPLO ABAIXO

                
                    //declarando a função
                    function algumaTarefa(PARÂMETRO1, PARÂMETRO2 = "padrão", PARÂMETRO3) {
                        < bloco de código >
                    }

                    //chamando a função sem um dos parâmetros
                    //NÃO CONFUNDA, o padrão do parâmetro2 não substituira na ordem
                    //Então o item "oqueQueroNoParâmetro3" será passado para o PARÂMETRO2
                    algumaTarefa(oqueQueroNoParâmetro1, oqueQueroNoParâmetro3)

RETORNO DA FUNÇÃO

Serve para usarmos alguma variável que declarada dentro dessa função fora dela, ou seja mais pra frente no código.

                
                    //declarando a função
                    function algumaTarefa() {
                        < bloco de código >
                        return NOMEdaVARIAVEL
                    }

                    //chamando a função sem um dos parâmetros
