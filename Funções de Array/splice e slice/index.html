SPLICE

Significa "emendar" ou "costurar"

Substitui o array original

A sintaxe deste método é:

        
            array.splice(index[, deleteCount[, elemento1[, ...[, elementoN]]]])

Espera um primeiro parâmetro que é um index do array

O segundo é quantidade de elementos que queremos remover a partir deste index

Os próximos são os elementos que queremos adicionar no lugar

Apenas o primeiro parâmetros é "obrigatório"

SLICE

Significa "fatiar" ou "dividir"

Extrai uma parte do array sem alterar o array original

A sinxtaxe deste método é

        
            array.slice([begin[, end]])

Possui um primeiro parâmetro que é o index inicial

O segundo parâmetro é o index final

Retorna todos os elementos entre o elemento de index "begin" e o anterior ao elemento "end"
